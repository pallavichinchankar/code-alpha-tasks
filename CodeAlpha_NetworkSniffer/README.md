# Basic Network Sniffer

## Description
A basic network sniffer developed using Python and Scapy to capture
and analyze live network traffic.

## Features
- Captures TCP, UDP, ICMP packets
- Displays source & destination IP
- Shows protocol and payload

## Tools Used
- Python
- Scapy
